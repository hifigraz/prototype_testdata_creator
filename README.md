# Testdata Creation

Testdata creation:

Usage: 

testdata_creator [-h|--help] (-c <connection_string> | --connection-string=<connection-string>) [-v|--verbose] [-n <number_of_persons> | --number=<number_of_persons>

     -h --help                      : display this help message and exit.
     -v --verbose                   : increase verbosity.
     -c --connection-string=<value> : connection string for database to use.
     -n --number=<value>            : number of persons to create
